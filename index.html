Real-Time Roller Coaster Safety System

This project demonstrates a real-time embedded system built using FreeRTOS on an ESP32. The system simulates a roller coaster station where safety is critical before dispatch.

The system monitors:

  • Harness safety
  • Coaster arrival detection
  • Emergency stop button

LEDs indicate system state:
- Harness secured
- Brakes engaged
- Emergency alert
- System ready for dispatch
