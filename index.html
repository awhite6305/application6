Real-Time Roller Coaster Safety System

This system simulates a safety-critical roller coaster control station using real-time embedded design principles. Multiple FreeRTOS tasks monitor harness status, coaster position, and emergency stop inputs. The system ensures that the ride can only be dispatched when all safety conditions are satisfied, demonstrating coordination, synchronization, and deterministic behavior in a real-time system.

The system monitors:

  • Harness safety
  • Coaster arrival detection
  • Emergency stop button

LEDs indicate system state:
- Harness secured
- Brakes engaged
- Emergency alert
- System ready for dispatch
